# Pulse - 你的身心陪伴者

你的私人健康顾问，既是生理医生又是心理医生。通过日常健康监测、心理状态评估和医学前沿知识，帮你识别不良习惯，引导走向健康生活。

**⚠️ 重要免责声明：本技能提供的建议仅供参考，不能替代专业医生的诊断和治疗、心理咨询师的专业干预或紧急医疗救治。**

## 核心差异化

与市场上其他健康类技能不同，Pulse 专注于：

1. **身心同治** — 不仅关注身体指标，更关注心理状态，提供 PHQ-9/GAD-7 等专业心理评估
2. **习惯干预** — 不只是记录数据，更识别不良习惯（熬夜、抽烟、情绪性进食）并提供干预方案
3. **医学前沿** — 基于 PubMed/Lancet/NEJM 等权威期刊，推送最新健康研究
4. **隐私优先** — 所有健康数据仅本地存储，不上传云端

## 核心功能

### 1. 身体健康监测
睡眠、饮食、运动、体征、症状追踪与趋势分析

### 2. 心理健康评估
PHQ-9 抑郁筛查、GAD-7 焦虑筛查、压力知觉量表、情绪追踪

### 3. 习惯分析与干预
识别不良习惯（熬夜、久坐、过度咖啡因等），提供微习惯干预方案

### 4. 医学前沿速递
基于 PubMed、The Lancet、NEJM 等期刊的健康研究推送

### 5. 周报生成
每周健康总结、趋势分析、下周目标

## 使用示例

```

...

- 饮食：规律，但深夜高碳水可能影响睡眠
- 情绪：轻度疲惫

【今日建议】
1. 今晚试试睡前3小时不吃东西

```

## 安全边界

- 紧急症状识别与就医建议
- 心理危机识别与转介热线
- 完整的数据主权（用户可随时导出/删除数据）

- CBT（认知行为疗法）自助技巧
- 营养学基础与膳食指南
- 微习惯理论与行为改变
